一、科学技术原理

1.1 CAPM和beta系数

资本资产定价模型（CAPM）是由美国学者威廉·夏普（William Sharpe）、林特尔（John Lintner）、特里诺（Jack Treynor）和莫辛（Jan Mossin）等人于1964年在资产组合理论和资本市场理论的基础上发展起来的，主要研究证券市场中资产的预期收益率与风险资产之间的关系，以及均衡价格是如何形成的，是现代金融市场价格理论的支柱，广泛应用于投资决策和公司理财领域。

资本资产定价模型的具体形式为：

(1)E(Ri)−Rf=βi(E(Rm)−Rf)

其中，E(Ri)表示资产i的期望收益率，Rf为无风险收益率（如银行存款利率，一般以一年期或三年期代表），E(Rm)表示市场期望收益率（通常用有具有代表性的指数的收益率来表示，比如沪深300指数），βi表示资产i的beta系数。

beta系数是金融领域中用于衡量资产风险和波动性的系数。该系数的定义式如下：

(2)βi=Cov(Ri,Rm)Var(Rm)
1.2 投资者风险偏好指数

A.D.Persaud在研究货币市场时发明了一种度量投资者情绪的良好方法：风险偏好指数（Risk Appetite Index）。其基本方法是度量资产的风险（即beta系数）和收益之间的相关性。

本项目在借鉴了该模型的基础上，根据市场上交易量与市场变化的关系，交易量也是反映市场风险偏好的良好指标，将其一并纳入本模型的考虑范围，研究三项指标之间的相关性。

1.2.1 相关性分析

一般而言，分析有序变量 (xi,yi),i=1,…,n 相关性的方式有两种：Pearson矩相关系数和Spearman秩相关系数等。Pearson矩相关系数通常适用于有明显线性关系的变量，或者双变量正态分布的数据，Spearman适用于等级数据。

Pearson矩相关系数有这样的局限性：第一，只能度量线性相关性；第二，对于非双变量正态分布的数据，不相关性和独立性并非等价，会产生错误的结果。所以本实验中使用Spearman秩相关系数。

Spearman秩相关系数是一种非参数统计量，度量两组有序变量(xi,yi),i=1,…,n之间的等级相关程度与相关的方向。简单说，就是将向量按照xi按照某种原则排序之后，yi与xi的排序保持一致性的程度和方向。设xi在{x1,…,xn}中是第Rxi小的，yi在{y1,…,yn}中是第Ryi小的，则有秩相关系数ρ计算公式如下：

(3)ρ=∑i=1n((Rxi−Rxi¯)(Ryi−Ryi¯))∑i=1n((Rxi−Rxi¯)2(Ryi−Ryi¯)2)

英国统计学家C.E.Spearman从上式中推导出简便的计算式，记为Spearman秩相关系数：

(4)ρs=1−6∑i=1n(Rxi−Ryi)2n(n2−1)

Spearman秩相关系数有这样的性质：

  1. 若两个变量排序完全相同，则Spearman秩相关系数为1；
  2. 若两个变量排序完全相反，则Spearman秩相关系数为-1；
  3. 若Spearman秩相关系数在(−1,1)之间，绝对值越大则排序程度越大；
  4. 若两个变量完全独立，则Spearman秩相关系数期望为0。

所以以ρs为检验统计量，拒绝域为{|ρs|>τ}。根据显著性水平α可以计算出临界值τ的值。

1.2.3 投资者风险偏好指数构建

本实验中，以同时期各行业周收益率、beta系数、周交易量中的任意两个之间计算得出的Spearman秩相关系数，作为衡量投资者风险偏好程度的风险偏好指数。

  1. 当风险偏好指数接近1时，说明投资者风险偏好高，市场情绪乐观，可以预测指数将会上涨；
  2. 当风险偏好指数接近-1时，说明投资者风险偏好低，市场情绪悲观，可以预测指数将会下跌。
  3. 进行显著性检验，作出上涨/下跌的判断。

通过将模型用于往年数据（如2005年至2023年）可以验证各个风险偏好指数模型的好坏。[1]

二、设计方案

2.1 数据来源和导入

本实验使用31个申万一级行业指数作为各行业数据（来源：申万宏源证券资讯网[4]），沪深300指数作为市场数据（来源：网易财经[5]）。

将各数据转换为csv格式，使用Python的csv库将每个交易日的日期、收盘指数、交易量读入到RiskAppetite类中。

2.2 收益率和beta系数的计算

设第i个申万一级行业第j周最后一天的收盘指数是di,j，则周收益率的计算公式是：

(5)fi,j=di,j−di−1,jdi−1,j

沪深300指数周收益率同理计算得到fm,j。

因为周收益率需要参与大量计算，所以此处使用了numpy.array来存储全部32组的周收益率数据以加快运算速度。

beta系数是反映一项资产风险的长期数据，所以参与beta系数计算的数据必须涵盖比较长的时间。设定一个时间窗口w周（本实验中默认为100周），可以计算出第i个申万一级行业第j周的beta系数：

(6)βi,j=1w−1∑k=j−w+1j(fi,k−fi,k¯)(fm,k−fm,k¯)1w−1∑k=j−w+1j(fm,k−fm,k¯)2

计算beta系数的分子是fi,k和fm,k的协方差，分母是fm,k的方差，可以直接使用numpy的cov函数求出协方差矩阵来计算。

记第i个申万一级行业第j周的总交易量为vi.j，是直接从数据中取出的。

2.3 投资者风险偏好指数的计算

RiskAppetite类中实现了函数RiskAppetiteIndex(self, date, type)，其中参数date指示了想要查看风险偏好指数的日期，type的值从以下枚举类中取出，指示了选择的风险偏好指数的类型。

xxxxxxxxxx
class RiskAppetiteType(Enum):

    VOLUME_YIELD = 2 # 交易量-收益率

该函数使用scipy中的spearmanr()方法，对于风险偏好指数类型对应的两组数据求出spearman秩相关系数和p值。

RiskAppetite类中实现了函数TrendSignal(self, date, type)，使用p值与显著性水平进行比较，可以进行假设检验。spearmanr()方法可以得出p值，直接用p值与α进行比较。如果p值较小，则说明可以认为有多空信号，根据风险偏好指数的正负性判定是看多信号还是看空信号。如果假设检验得出结论“可以认为两组数据相关”，则根据spearman秩相关系数的正负性，判断发出看多或看空信号。

三、创新性描述

  1. 参考的资料中的风险偏好指数仅考虑收益（收益率）和风险（beta系数）之间的相关性。然而，通过往年数据可以看出，市场的波动与交易量也有很大的关系。交易量越大，说明投资者投资热情越大，市场交易很火热，收益和风险都同步增加。无论是2008年金融危机还是2015年股灾，都能在其之前几个月看到暴增的交易量。所以本实验创新性地将交易量也考虑进来，试图寻找交易量与市场波动的潜在关系。有了多个模型，也提供了将多个模型结合使用的可能性。
  2. 能够找到的最新的有关风险偏好指数的研究是在2014年，在2015年股灾之前。在现在来对这个模型进行研究，可以更准确地衡量这一模型在面对大波动时的稳定性。稳定性是投资领域非常重要的一个评价标准。
  3. 程序中的所有参数都可以调节（包括开始和结束的时间、beta系数计算时间窗、显著性水平等），具有灵活性。
  4. 程序的效率瓶颈用numpy优化，实测计算效率相当高，所有除去图像渲染的工作都可以在几秒内完成。
  5. 输入部分较为灵活，可以处理从各种渠道下载的其他csv文件，可以处理除了沪深300、申万一级行业指数以外的数据。
  6. 400余行程序全部由自己完成。

结果分析

最终的结果说明：

  1. 比起原始的风险偏好指数，涉及交易量的扩展风险偏好指数在实际交易中的表现并不优良。表明，虽然交易量与市场的走势有关系，但是并不能通过其与风险或收益的行业间相关性来衡量。
  2. 由基础数据表中的第二张图（VOLUME_BETA Model Basic Graph）可以看出，交易量与风险（Beta指数）有着很强的相关性，而且随着时间变化，方向也会变化：有多段时间都是正相关，在少数的短时段负相关。而中间最长的一段持续强正相关的时间，从2010年一直持续到2017年。这是一个很有意义的发现。
  3. 在2014年的研究中，原始的风险偏好指数的表现是很好的，在本次实验中也成功地复现了：取得了3536%的收益，并且最大回撤只有11.23%，而且在面对股灾的时候也没有造成很大损失。但是这个模型在后期（2019年）表现很差，在高处做多、低处做空，这一次回撤达到了43.26%，最终的总收益率（到2023年）也只有153%。这说明该模型的稳定性有待商榷，必须由人员时刻监控，不能让该模型自行交易。
